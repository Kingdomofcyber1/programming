ملخص بايثون الشامل للامتحان

الرموز الرئيسية في بايثون

الرموز هي الأدوات الأساسية لكتابة الكود. إليك أهم العوامل والكلمات المفتاحية:

العوامل الحسابية والتخصيص

  • +: الجمع (مثال: 5 + 3 = 8)
  • -: الطرح (مثال: 5 - 3 = 2)
  • *: الضرب (مثال: 5 * 3 = 15)
  • /: القسمة (مثال: 5 / 2 = 2.5)
  • //: القسمة الصحيحة (مثال: 5 // 2 = 2)
  • %: الباقي (مثال: 5 % 2 = 1)
  • **: الأس (مثال: 2 ** 3 = 8)
  • +=: زيادة القيمة (مثال: x += 1 يعني x = x + 1)
  • -=: نقصان القيمة (مثال: x -= 2)

عوامل المقارنة والمنطقية

  • ==: المساواة (مثال: 7 == 5 → False)
  • !=: عدم المساواة (مثال: 7 != 5 → True)
  • <, >, <=, >=: مقارنة (مثال: i < 5)
  • and: كلا الشرطين صحيحان (مثال: True and False = False)
  • or: أحد الشرطين صحيح (مثال: True or False = True)
  • not: عكس القيمة (مثال: not True = False)

كلمات التحكم والرموز الأخرى

  • for: تكرار على تسلسل (مثال: range(5))
  • while: تكرار حسب شرط
  • break: إيقاف الحلقة
  • continue: تخطي التكرار الحالي
  • pass: لا تفعل شيئًا
  • else: مع الحلقات أو الشروط
  • input(): إدخال من المستخدم
  • print(): إخراج البيانات
  • #: تعليق
  • .py: امتداد ملف بايثون

جدول ملخص الرموز

الرمز الاستخدام مثال
+= زيادة القيمة x += 1 → x = x + 1
% الباقي 5 % 2 = 1
== المساواة 7 == 5 → False
break إيقاف الحلقة if i == 2: break
continue تخطي التكرار if i % 2 == 0: continue

المفاهيم الأساسية

هذه المفاهيم هي أساس البرمجة في بايثون وتظهر بكثرة في الامتحانات.

المتغيرات وأنواع البيانات

  • int: عدد صحيح (مثال: 5)
  • float: عدد عشري (مثال: 5.5)
  • str: نص (مثال: "Hello")
  • bool: قيمة منطقية (True, False)
  • قواعد التسمية: تبدأ بحرف أو _، لا تحتوي على رموز مثل # أو %.
x = 10 # int
y = 3.14 # float
name = "Ali" # str
is_valid = True # bool

حلقة For

تُستخدم للتكرار على تسلسل محدد (مثل range()).

for i in range(2, 10, 2): # من 2 إلى 8 بخطوة 2
print(i)
# الناتج: 2, 4, 6, 8

حلقة While

تُنفذ طالما الشرط صحيح.

i = 0
while i < 3:
print(i)
i += 1
# الناتج: 0, 1, 2

الشروط

تُستخدم لاتخاذ قرارات بناءً على شروط.

x = 5
if x % 2 == 0:
print("زوجي")
else:
print("فردي")
# الناتج: فردي

الإدخال والإخراج

لتجنب EOFError، استخدمي try-except.

try:
name = input("أدخل اسمك: ")
print("مرحبًا", name)
except EOFError:
print("لا يوجد إدخال")
name = "ضيف"
print("مرحبًا", name)

بايثون كمترجم

بايثون مترجم، ينفذ الكود سطرًا بسطر، وحساس للحروف.

أمثلة عملية

مثال 1: حلقة While

i = 0
while i < 3:
print(i)
i += 1
# الناتج: 0, 1, 2

مثال 2: حلقة For مع Continue

for i in range(4):
if i % 2 == 0:
continue
print(i)
# الناتج: 1, 3

مثال 3: مقارنة

7 == 5 → False

نصائح للامتحان

راجعي الرموز: احفظي +=, %, break.

تتبعي الكود: اكتبي قيم المتغيرات يدويًا.
